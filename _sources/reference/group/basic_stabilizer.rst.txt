.. class:: template<typename Alloc, typename DupChecker = DupCheckerNop> basic_stabilizer

    A  `Stabilizer` type which filters permutations from a base group
    simply by whether they fix a given element.
    It may therefore represent only a subgroup of the stabilizer it is
    defined as.
