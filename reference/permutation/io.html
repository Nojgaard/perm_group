Returns:a list of cycles of non-negative numbers parsed from the given stream. It does not necessarily represent a permutation (see the other overload).

void read_permutation_cycles(const std::string &s, Perm &p)¶

Parse a permutation in cycle notation from a stream or a string (s) and store it in p. Permutations are zero-indexed.

The overloads without n requires SizeAwarePermutation<Perm>, and sets n = perm_group::degree(p).

Throws:io_error if parsing fails.
Throws:io_error if a number is n or higher.
Throws:io_error if the data does not represent a permutation.
