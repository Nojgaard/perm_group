.. function:: template<typename Perm> \

              template<typename Perm> \
              void read_permutation_cycles(const std::string &s, Perm &p)

    Parse a permutation in cycle notation from a stream or a string (`s`) and store it in `p`.
    Permutations are zero-indexed.

    The overloads without `n` requires `SizeAwarePermutation<Perm>`,
    and sets `n = perm_group::degree(p)`.

    :throws: `io_error` if parsing fails.
    :throws: `io_error` if a number is `n` or higher.
    :throws: `io_error` if the data does not represent a permutation.
